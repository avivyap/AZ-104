VM = Virtual Machine = Infraestructura como Servicio (IaaS)

Si queremos conectarnos por ejemplo a un windows server 2025 y la maquina tiene un tamaño ridículo, de 1GB, nos va a dar problemas a la hora de conectarnos ya que un windows server necesita mas recursos. Lo que podemos hacer para solucionar este problemas es irnos a "Disponibilidad y escala" y aumentar el tamaño de la maquina
