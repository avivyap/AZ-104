Con SaaS, básicamente se alquila o se usa una aplicación totalmente desarrollada.

Software como servicio (SaaS) es el modelo de servicio en la nube más completo desde el punto de vista del producto.

Aunque el modelo de SaaS puede ser el menos flexible, también es el más fácil de ponerse en marcha. Requiere la menor cantidad de conocimientos técnicos o experiencia para emplearlo plenamente.
