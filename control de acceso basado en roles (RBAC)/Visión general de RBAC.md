RBAC = Control de acceso basado en roles/permisos

- RBAC en azure es el sistema que usamos para autorizar el acceso de recursos

- Administra accesos específicos a los recursos de Azure 

- Define que acciones se pueden realizar 

Un rol se puede aplicar a distintos niveles:

- Grupo de administración
- Suscripción
- Grupo de recursos
- Recurso individual

Cuando asignas un rol no estas dando acceso total al recurso sino permiso para ejecutar ciertas acciones concretas
### Herencia 

**Un rol definido en un nivel superior aplica a todos los niveles inferiores**
