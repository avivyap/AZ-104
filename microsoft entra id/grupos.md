Nos permiten agrupar y organizar usuarios y permisos, para no tener que aplicar uno a uno para poder aplicarle x permisos


### Tipos

Grupos de seguridad, nos permite controlar el acceso a recursos compartidos, por ejemplo (apps, VMs,etc), asignan permisos segun politicas, pueden incluir usuarios y dispositivos y requieren administración por un admin de Entra ID

Grupos de Microsoft 365, usados para la parte de colaboración y productividad, se proporciona acceso a buzon, calendario compartido, teams y M365, nos permiten colaboración con usuarios externos

Ahi tenemos los tipos de pertenencia

- Asignada <- Forma manual
- Usuario/Dispositivo dinámico <- Forma automática dependiendo del usuario/dispositivo 

**Lo normal es que cada usuario tenga un rol dependiendo del grupo al que pertenece, NO es recomendable asignar un rol a un único usuario**
