### Propiedades

En la parte de propiedades, definimos a quien se le aplica el restablecimiento de contraseña 

![[Pasted image 20260703115920.png]]

Ninguno <- Nadie usar el auto servicio
Seleccionado <- Un grupo concreto 
Todos <- Todos los usuarios de la organización (Los admins siempre lo tienen habilitado)
### Métodos de autenticación

![[Pasted image 20260703120036.png]]

En esta pestaña definimos cuantos métodos de autenticación necesitamos para que un usuario pueda restablecer su contraseña 

### Registro

![[Pasted image 20260703120135.png]]

En la parte de registro es donde se configura si los usuarios pueden registrase para el SSPR al iniciar sesión 

Si ponemos que "Si" al iniciar sesión se les pedirá a los usuarios que configuren sus métodos de inicio de sesión

Y podemos hacer que cada x días tengan que confirmar sus datos
