Bicep es un lenguaje que usa una sintaxis declarativa para implementar recursos de Azure 

### Características

- Simplifica y mejora la legibilidad de las plantillas ARM, permite reutilizar código con módulos 

- Cuando lanzamos una plantilla bicep, esta se va a convertir automáticamente en una plantilla JSON ARM compatible 

- Facilita la automatización y el versionado de infraestructuras 

### Conversión

Este comando te permite convertir una plantilla Bicep en ARM JSON

	bicep build
