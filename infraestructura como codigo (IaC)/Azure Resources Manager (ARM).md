Proporciona una capa de gestión que permite:

- Crear recursos 
- Actualizar recursos
- Eliminar recursos

Facilita la organización y control de todos los recursos

### Plantillas 

ARM permite gestionar y orquestar recursos en Azure de manera centralizada

Utiliza plantillas declarativas en JSON para describir recursos y configuración

Facilita la creación, actualización y eliminación de recursos de forma controlada 

Si importamos una plantilla en azure, NO se despliega el servicio automáticamente, sino que se almacena y después tienes que desplegarlo
